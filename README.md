# 🔐 Tela de Login

Este é um projeto simples de uma tela de login moderna, desenvolvido com foco em prática de HTML, CSS e conceitos de design de interfaces.

## 🚀 Tecnologias utilizadas
- HTML5
- CSS3

## 🎨 Funcionalidades
- Interface de login responsiva
- Design moderno com efeito de transparência (glassmorphism)
- Campos de entrada estilizados (email e senha)
- Botão de login com efeitos visuais

## 📚 Objetivo
Este projeto foi desenvolvido com o objetivo de praticar a criação de interfaces web e aprimorar habilidades em estruturação com HTML e estilização com CSS.

## 💡 Melhorias futuras
- Integração com backend (autenticação real)
- Validação de formulário com JavaScript
- Responsividade completa para dispositivos móveis
